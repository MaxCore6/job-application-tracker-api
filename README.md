# job-application-tracker-api


Backend-сервис для структурированного учёта откликов на вакансии, этапов собеседований и их истории во времени, предназначенный для личного контроля процесса поиска работы.

## Problem 
При большом количестве откликов (20-50+) становится сложно отслеживать, на каком этапе находится каждая вакансия, кто уже ответил, кто отказал и сколько времени занимает процесс найма.

## Solution 

Сервис предоставляет API для хранения откликов, управления их статусами и сохранения истории изменений, позволяя пользователю видеть полный жизненный цикл каждой заявки. Сервис моделирует процесс найма как набор связанных сущностей и состояний

## Business Value

- Сокращение времени обработки
- Прозрачный контроль воронки найма
- Снижение когнитивной нагрузки при большом количестве откликов
- Чёткая история взаимодействий с компаниями
